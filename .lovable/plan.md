

## תוכנית: הוספת מצב "הפיכת צבעים" (Light Mode) לתפריט הנגישות

### מה ייעשה
הוספת אפשרות חדשה בתפריט הנגישות בשם **"הפיכת צבעים"** שתהפוך את כל האתר מערכת צבעים כהה (שחור + ירוק) לערכת צבעים בהירה (לבן + ירוק), כולל התאמת צבעי גופנים, כרטיסים, רקעים, גבולות ועוד.

### שינויים נדרשים

**1. `src/components/AccessibilityMenu.tsx`**
- הוספת שדה `invertColors: boolean` ל-`AccessibilitySettings`
- הוספת `invertColors: false` ל-defaults
- הוספת toggle class `a11y-invert-colors` ב-useEffect
- הוספת פריט תפריט חדש "הפיכת צבעים"

**2. `src/index.css`**
- הוספת class חדש `.a11y-invert-colors` שידרוס את כל משתני ה-CSS לגרסה בהירה:
  - רקע: לבן (0 0% 98%)
  - טקסט: כהה (0 0% 10%)
  - כרטיסים: לבן/אפור בהיר
  - גבולות: אפור בהיר
  - Primary נשאר ירוק (168 45% 30%)
  - שינוי card-gradient, glass-bg, shadows לגרסאות בהירות
  - התאמת sidebar, muted, secondary, popover לצבעים בהירים
  - big-cursor יתהפך לסמן כהה על רקע בהיר

### פרטים טכניים
הגישה: דריסת CSS variables באמצעות `!important` על class ב-`:root`, בדיוק כמו שמימושי הנגישות הקיימים עובדים (למשל `a11y-high-contrast`). זה מבטיח שהשינוי חל על כל הדפים והקומפוננטות באתר ללא צורך בשינוי קומפוננטות בודדות.
